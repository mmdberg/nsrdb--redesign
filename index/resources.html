Meteorological Statistical Model 3 (MTS3)

The National Solar Radiation Database User's Manual (1991–2010 Update) describes the third version of the NSRDB. The manual provides guidance on using database products, acquaints the user with the many features of the database update, and provides information about how to use database products.

Meteorological Statistical Model 1 (MTS1)

The National Solar Radiation Database User's Manual (1961–1990) describes the first version of the NSRDB. The manual provides guidance on using database products, acquaints the user with the many features of the database, and provides information about how to use database products.

LINKS

The following resources supplement and provide guidance for using NRSDB data.

  1. December 8, 2015 Webinar Recording: Video recording of a recent webinar describing the latest release of the NSRDB
  2. December 8, 2015 Webinar PDF: PDF of slides from a recent webinar describing the latest release of the NSRDB
  3. Best Practices Handbook for the Collection and Use of Solar Resource Data for Solar Energy Applications: An updated guide for the solar energy industry on the key aspects of solar resource characterization
  4. Resource Assessment & Forecasting: NREL's Resource Assessment & Forecasting group website for renewable energy resource measurements, models, maps, and support services
  5. Measurement and Instrumentation Data Center (MIDC): Irradiance and meteorological data from stations around the United States
  6. Renewable Resource Data Center (RReDC): Historical irradiance data sets and documentation for sites in the United States, including solar models, algorithms, and calculators
